# SIMP - SImple Music Player
TUI music player that only play music.

currently it can only play single music. future goals is to be able to play and list musics in directory.

```
SIMP - SImple Music Player

USAGE:
    simp FILENAME
    simp OPTION
OPTION:
    -v, --version
    show current version
```

## Installation
```
$ git clone 'http://github.com/sevaaadev/simp'
```
dependencies:
- lib-ncursesw
```
$ make
```
then you can put the binary to your path directory

## About this project
i was curious to know how sound is represented digitally, thus i created this music player for me to learn about digital audio. this is a toy project but i will try to polish it until its usable for simple usage.
the goals in mind are:
- ability to do playback
- list music in directory and be able to choose to play
- able to modify the queue
